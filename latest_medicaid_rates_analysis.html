Data Sources & Methodology

🏥 Provider Type Classification

  • Small Group: Independent groups with 1-9 physicians
  • Medium Group: Independent groups with 10-99 physicians
  • Large Group: Independent groups with >100 physicians
  • Health System: Providers affiliated with health systems via AHRQ linkage datasets
  • Overall: All providers combined regardless of affiliation

📋 Data Source Reference

  • T-MSIS Medicaid claims (opendata.hhs.gov)
    Purpose: Provider-level payment data aggregated by NPI × HCPCS × month
    Filters Applied: HCPCS Code 99214 (office visits only), Avg payment ≥$15 (excludes dual-eligible cost-sharing), ≥12 claims per provider (minimum volume), 2023-2024 data only
  • Payer machine-readable files (MRF)
    Purpose: Links individual providers (NPIs) to organizations (TINs) with practice size data
  • AHRQ Compendium of U.S. Health Systems
    Purpose: Identifies health systems and their member practices via PAC IDs
  • Mimilabs derived table (70% CMS/PECOS data, 30% payer MRF and state records)
    Purpose: Multi-source crosswalk linking NPIs to organizational identifiers (PAC IDs, CCNs, etc.)

Data Limitations:

  • AHRQ affiliation data is snapshot-based (~2018-2023 vintage) and may miss informal partnerships. Health system status determined by formal ownership/management relationships only.
  • Medicaid provider spending data includes all Medicaid delivery models (Fee-for-Service, Managed Care Organizations, Primary Care Case Management, etc.). The underlying dataset does not allow filtering by specific payer or delivery model - payments represent blended rates across all Medicaid programs within each state.
