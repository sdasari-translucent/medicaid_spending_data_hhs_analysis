# Translucent Mimilabs Analysis

Comprehensive healthcare reimbursement analysis using Mimilabs MCP integration. Analyzes Medicaid payment patterns across states and provider types to identify contracting opportunities and market positioning strategies.

## Usage

```bash
# Install dependencies
pip install -r requirements.txt

# View the analysis
python -m http.server
# Then open http://localhost:8000/latest_medicaid_rates_analysis.html in your browser
```

### Requirements

See `requirements.txt` for Python dependencies:
- pandas
- matplotlib
- seaborn
- numpy
